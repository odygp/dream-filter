# Dream Rig

Adjustment-layer playground for tuning dreamy reveal/hide effects on text and images,
built for scrollytelling pages. Will live at https://dream-filter.holy.black.

- **Text mode** — Blur, Drift, Warp (SVG turbulence), Glow, and Ghost layers with per-word stagger
- **Image mode** — Blur, Drift, Warp, Veil, and Bloom layers; upload your own image (never leaves the browser)
- **Presets** per media type, a scroll demo that simulates the scrollytelling in/hold/out
- **Export** — generates vanilla CSS + JS with the tuned values baked in, with
  IntersectionObserver and GSAP ScrollTrigger drivers

Single static file — no build step. Open `index.html` or deploy anywhere static.
